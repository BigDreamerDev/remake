# Lighthouse Irregularity SW-MAR-4020

station: The Unlit Tower
keeper: Keel
review-cycle: low tide

## Beacon Behaviour

The beacon emitted 9 flashes during a period when power to the lamp was confirmed
disconnected. Observers at sea reported the flashes as visible only when looking away from
the tower. Inland observers reported the light reflecting from puddles inside locked rooms.
A wet handprint appeared on the inside of the lantern glass at a height requiring no known
ladder.
## Keeper Instructions

- No bell is to be answered from below deck.
- If a hallway smells of kelp, count the doors but do not open the third.
- Mirrors facing standing water must be covered with cloth marked at all four corners.
- Do not record your own name in a wet logbook.
- If a window shows open sea where a wall is expected, draw the curtain and file Form MW-13.

## Closing Note

This file should be used to increase atmospheric uncertainty around coastal records. It
should not be treated as a navigational chart, solution index, or command reference.
