# 1666 Poor-Law Return — Fire Damage Return

**Accession:** SW-ARC-VLM-1666-0162
**Custody:** Returned Box Registry
**Current Location:** Cresswell Quay
**Material:** photographic tissue
**Hand:** copperplate
**Status:** sealed pending comparison

## Catalogue Note

The item was received as a poor-law return and is treated as a historical document for the purposes of Codicology review. The recorded date is 1666, though the document has been known to prefer adjacent years when stored in damp shelving.

Condition on intake: paper darkens when read aloud.

The document should not be cleaned, flattened, warmed, sung near, or compared directly against any recollection supplied by the donor.

## Historical Summary

A historical seal appears in the correct position, but the impression is reversed, as if pressed from beneath the page.

The document references Mossgate as if it were a civic location rather than an internal Foundation designation.

A docket note refers to the blank visitor entry. This may be administrative noise.

Two folios use the same page number. Both have been treated as the original for different audits.

## Handling

- If the room number appears in pencil, do not erase it.
- File copies separately from originals even when the original insists there is no copy.
- If the record becomes heavier after transcription, stop weighing it.
- Do not move this item to a brighter shelf to prove that the ink is ordinary.
