# 01 — Word Order and the Verb Cluster

## Word order: Subject — Object — Verb

The Pre-Threshold Tongue places the verb at the **end** of the clause. Any objects and modifiers come **before** the verb.

| English | Senion |
|---|---|
| I see the door. | **Ya kor osh-nai.** |
| The witness reads the record. | **Osher kelhom thuri-or-nai.** |
| The archive holds the name. | **Arvel meren dom-or-seln.** |

The final word in a Senion clause is almost always the verb (with its suffixes). When in doubt about where the clause ends, look for the **evidential**.

## The verb cluster

Senion verbs build outward from a root in a fixed order. Each slot is optional **except the evidential**.

```
ROOT  ( -ASPECT )  ( -TENSE )  ( -MOOD )  -EVIDENTIAL
```

The five slots:

| Slot | Markers | What it does |
|---|---|---|
| **Root** | `han`, `osh`, `morh`, `thuri`, `kelh`, `nadi`, `glosi`, `par`, `dom`, … | The action or state. |
| **Aspect** | `-or`, `-us` | `-or`: ongoing / habitual (imperfective). `-us`: completed (perfective). May be omitted. |
| **Tense** | `-ren`, `-thel` | `-ren`: past. `-thel`: future. Omitted for present/habitual. |
| **Mood** | `-vor` | Sealed / restricted from repetition. Use sparingly. |
| **Evidential** | `-nai`, `-nal`, `-thur`, `-mar`, `-kel`, `-sai`, `-seln` | The source of the speaker's knowledge. **Mandatory.** |

### Worked examples

| English | Senion | Cluster |
|---|---|---|
| The door is open. (I see it) | **Kor han-nai.** | `han + -nai` |
| The door is open. (The record says so) | **Kor han-seln.** | `han + -seln` |
| The door opens. (Ongoing, I see it) | **Kor morh-or-nai.** | `morh + -or + -nai` |
| The door opened. (Completed past, I saw it) | **Kor morh-us-ren-nai.** | `morh + -us + -ren + -nai` |
| The door will open. (Sealed; the record says so) | **Kor morh-thel-vor-seln.** | `morh + -thel + -vor + -seln` |

The order of the suffixes never changes. A clause with `-vor` *after* the evidential is ungrammatical.

## Postposition placement

What English does with **prepositions**, Senion does with **postpositions**. They attach to the noun they govern with a hyphen.

| English | Senion |
|---|---|
| in the archive | **arvel-eth** |
| to the door | **kor-toh** |
| from the building | **vehel-aru** |
| of the rank | **tier-en** |

A noun with a postposition is still a noun phrase, and acts as a modifier of the clause. Postpositional phrases generally appear **before** the verb, after the subject and object.

```
SUBJECT     (OBJECT)   (POSTPOSITIONAL PHRASE)   VERB-CLUSTER
Osher       kelhom     arvel-eth                  thuri-or-nai
witness     record     in-the-archive             reads
```

## Plurals

Number is **unmarked** unless context requires it. *kor* may mean *door* or *doors* indifferently. To force a plural reading, append **`-lan`**: *korlan*, *doors specifically*.

```
kor             door / doors (ambiguous)
korlan          doors (definitely plural)
thren korlan    three doors
```

## Articles

There are none. Definiteness is inferred from context. Where ambiguity must be resolved, the demonstratives **ha** (*this / here*) and **to** (*that / there*) are used as bare nouns or prefixed onto the noun.
