# Senionsoln — Direct Reference

**Reference:** SW-COD-SEN-DIR / Bcy 28, 3192 YOLD
**Working group:** Codicology
**Track:** Direct (structured)
**Alternate track:** *Discovery* (see `handbook/codicology/senion_discovery/`)

This packet is the **structured** introduction to the Pre-Threshold Tongue. It tells you the grammar and the vocabulary directly. The alternate Discovery track requires you to reconstruct the language from a stone; this one does not.

Both tracks lead to the same verification gate (`senion.verify`). A staff member is free to use whichever pack they find more efficient. The working group records which pack each verified staff member used; correlations are not currently published.

## What the Pre-Threshold Tongue is

**Senionsoln** (also: *the Senion Tongue*, *the Pre-Threshold Tongue*) is the language attested in the prefatory matter of the Pre-Threshold registers held in the lower stacks. It is not a historical natural language presently recognised by the Foundation, but is consistent enough across the tradition to be catalogued as a single tongue. The earliest surviving register provides what appears to be the scribes' own designation: *Senionsoln*.

A working reconstruction of the language exists. This packet is the digest.

## What you need to know in five lines

1. **Word order is Subject — Object — Verb.** The verb comes last. The verb carries everything that matters.
2. **Every declarative sentence ends in an evidential.** The evidential states *how the speaker knows* — witnessed, heard, inferred, recorded, etc.
3. **There are no articles** (*a*, *an*, *the*). Number is unmarked unless it matters.
4. **Prepositions are postpositions.** *in X* is *X-eth*, not *eth-X*.
5. **The sealed-mood particle `-vor`** marks a clause as restricted from repetition. It is *not* an evidential.

The rest of this packet expands those five points.

## Note on the canon

This packet reflects the **revised** Codicology reading (SOV; verb cluster final; numeral classifiers from a closed set). Earlier handbook entries (SW-HB-SEN-001 through 023) used **SVO** word order and a simpler verb. The verification gate is set to the revised reading. If you find archive material that uses the older reconstruction, treat it as historical and not authoritative.

The marker for the revised reading on Foundation documents is the watermark **SOV-CIR-**. The marker for the deprecated reading is **SVO-CIR**. If you see neither, treat the document as informal.
