## Historical Summary

The catalogue number was assigned before the accession arrived, according to the receipt book.

Two folios use the same page number. Both have been treated as the original for different audits.

The document references Saltmarsh as if it were a civic location rather than an internal Foundation designation.

The item has been requested by a reading room that is not currently open to staff.
