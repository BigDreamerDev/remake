## Catalogue Note

The item was received as a burial index and is treated as a historical document for the purposes of Codicology review. The recorded date is 1666, though the document has been known to prefer adjacent years when stored in damp shelving.

Condition on intake: seal impression wet to touch.

The document should not be cleaned, flattened, warmed, sung near, or compared directly against any recollection supplied by the donor.

## Historical Summary

Two folios use the same page number. Both have been treated as the original for different audits.

The catalogue number was assigned before the accession arrived, according to the receipt book.

A docket note refers to the lower switchboard. This may be administrative noise.

The document references Mossgate as if it were a civic location rather than an internal Foundation designation.
