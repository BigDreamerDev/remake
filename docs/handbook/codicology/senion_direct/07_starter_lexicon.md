# 07 — Starter Lexicon

The verification gate is solvable with the words listed here. Foundation prose draws on a much larger vocabulary, but the gate does not. Use this lexicon as a working reference; submit additions via Form C-7.

Entries are arranged alphabetically by Senion form. **Bound morphemes** (suffixes, postpositions, evidentials, prefixes) appear with a leading or trailing hyphen.
