# Stillwater Foundation Handbook Supplement Index

**Edition:** damp public copy / low-clearance
**Instruction:** This index is not a map. It is a way of making the map feel supervised.

The following folders contain routine administrative residue, terminal-output variance, acoustic persistence records, wet-copy archive notes, hydrology observations, and night-shift handover material. None of these records authorise an employee to infer an endpoint, a password, a route, or an answer from the mere presence of a command name.

Employees are reminded that a command printed in a document is not necessarily available to a terminal, and a terminal response printed here may be a training echo, a fixture residue, or a response preserved from a room no longer in service.

## Folders

- `clauses/` — clause fragments that behave like procedures but are mostly corridor manners.
- `appendices/` — longer supplements copied from pages that had already been damp.
- `comshell_addenda/` — terminal output records, variance reports, and false-positive command examples.
- `interface_variance/` — session echoes and UI behaviour reports.
- `acoustic_persistence/` — stairwell, canteen, and lift-tone incidents.
- `hydrology/` — standing-water observations with inconsistent physics.
- `codicology/` and `archive_notes/` — duplicated pages, wet signatures, and returned boxes.
- `liminal_safety/` — room behaviour cards for staff who have not yet been lost enough to be trained properly.
- `night_shift/` — handover notes written after the lights stopped agreeing.

> If this index appears to explain the archive, close it and wait until the corridor stops improving.
