# Expanded cross-reference sheet 090

Responsible office: Pattern Integrity Working Group
Observation site: mailroom return shelf
Clauses consulted: 16.8, 19.0, 4.4, 21.2

## Scope
Where the command appears to answer a question not asked, attach the transcript to the nearest related form and do not ask the terminal to clarify itself.

The training copy may contain deliberate duplicated text. Duplicate text must be preserved in order to prove the copy is not the original.

The operator should read the output only once. A second reading may produce a different room but the same filing requirement.

When a command returns silence, the silence should be treated as a complete sentence for the purpose of audit review.

Staff who experience nostalgia while reading a terminal output should record the sensation under routine environmental discomfort, not memory recall.
