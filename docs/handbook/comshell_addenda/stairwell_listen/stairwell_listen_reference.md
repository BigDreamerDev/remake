Responsible office: Hydrological Survey Unit.

## Purpose
`stairwell.listen` is recorded here as a low-clearance or compatibility command. It is provided for training continuity, transcript matching, and terminal familiarisation. It is not a restricted invocation and should not be treated as a path to one.

