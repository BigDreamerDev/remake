# Low-clearance terminal familiarisation packet 20

Issued by: Hydrological Survey Unit
Observation room: Floor -3 queue line

This packet is a training fragment for operators who need to recognise harmless terminal output without escalating every liminal phrase. It intentionally repeats wording from old logs so that staff can become bored enough to stop seeing patterns where there are only forms.

## `descale.status`
Expected or tolerated output includes:
- `Do not enter if you hear someone finishing your sentence.`
Repeated use during quiet hours may cause the prompt to appear further away than the keyboard.

## `memory.recall`
Expected or tolerated output includes:
- `Memory recall unavailable while the handbook is facing east.`
Do not use these outputs to infer restricted endpoints, personnel rank, or sealed room assignments.

## `memorial.check`
Expected or tolerated output includes:
- `You are not currently absent enough to be excused.`
Where a command returns contradictory states, staff should preserve both copies and avoid deciding which one is standing nearer to the door.

## `mailroom.check`
Expected or tolerated output includes:
- `One envelope contains only the sound of opening it.`
Repeated use during quiet hours may cause the prompt to appear further away than the keyboard.

## `seal.verify`
Expected or tolerated output includes:
- `seal is probably still deciding`
A blank output is not a failure condition. A blank output may be a successful output witnessed from behind.

## `badge.read`
Expected or tolerated output includes:
- `Magnetic stripe contains 0 approved corridors and 1 remembered corridor.`
Do not attempt to reconcile the output with eyewitness reports unless the eyewitness was present before the command was entered.
