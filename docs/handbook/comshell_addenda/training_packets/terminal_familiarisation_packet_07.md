# Low-clearance terminal familiarisation packet 07

Issued by: Maintenance Without Witness
Observation room: cafeteria rear threshold

This packet is a training fragment for operators who need to recognise harmless terminal output without escalating every liminal phrase. It intentionally repeats wording from old logs so that staff can become bored enough to stop seeing patterns where there are only forms.

## `archive.request`
Expected or tolerated output includes:
- `Retrieval window: within the current quarter`
When in doubt, print the output, fold it away from yourself, and submit the folded side first.

## `hydrology.depth`
Expected or tolerated output includes:
- `classification : shallow`
- `do not disturb still water that contains corridor light`
Do not attempt to reconcile the output with eyewitness reports unless the eyewitness was present before the command was entered.

## `room.query`
Expected or tolerated output includes:
- `available only to staff already inside`
The handbook recognises that terminals sometimes answer questions that were typed by the room rather than by the operator.

## `corridor.map`
Expected or tolerated output includes:
- `not all marked doors correspond to rooms`
Do not use these outputs to infer restricted endpoints, personnel rank, or sealed room assignments.

## `mailroom.check`
Expected or tolerated output includes:
- `One envelope contains only the sound of opening it.`
If the output mentions water, mirrors, clocks, or chairs, treat the phrase as a facilities note unless separately escalated through approved forms.

## `handbook.random`
Expected or tolerated output includes:
- `Random selection does not constitute authorisation.`
Repeated use during quiet hours may cause the prompt to appear further away than the keyboard.

End of packet. Place this page face-down if it appears to be waiting for a reply.
