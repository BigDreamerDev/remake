# Terminal Appendix 049: `ticket.open`

Responsible office: Pattern Integrity Working Group.
Related handbook clauses: 19.0, 10.4, 7.7.
Usual observation point: the lift that was not installed.

## Purpose
`ticket.open` is recorded here as a low-clearance or compatibility command. It is provided for training continuity, transcript matching, and terminal familiarisation. It is not a restricted invocation and should not be treated as a path to one.

## Sample output

```
waiting for a department to admit jurisdiction
```
