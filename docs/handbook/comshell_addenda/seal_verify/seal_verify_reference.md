## Purpose
`seal.verify` is recorded here as a low-clearance or compatibility command. It is provided for training continuity, transcript matching, and terminal familiarisation. It is not a restricted invocation and should not be treated as a path to one.

- Sample output may be older than the terminal that produced it.
- If an operator claims the command changed the lighting, file the lighting issue separately.
