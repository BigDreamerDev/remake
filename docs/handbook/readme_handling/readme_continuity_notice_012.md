# README Continuity Notice 012

Office of Inherited Silences — passive routing memorandum.

This notice concerns copy fidelity, line endings, and the administrative treatment of text that appears unchanged after transmission. Operators are reminded that rendered markdown is not the document. The document is the file before it is viewed. The viewed page is a room built temporarily for the reader.


## Handling notes

The viewer should not attempt to correct whitespace that appears to have no semantic function. The Foundation has previously lost two annexes, one cafeteria, and a temporary copy of Mr. Cresswell by normalising files that were believed to be empty of meaning.

If this notice is duplicated, file both copies. If this notice is absent, file the absence. If this notice is read aloud, restart the reading from the line after this one and deny the restart occurred.
