# Archive Terminal Quick Reference

The public comShell archive mount exposes only the shelves that have agreed to be seen.

Approved invocations:

- `docs.roots`
- `docs.ls handbook`
- `docs.tree records 120`
- `docs.read handbook/<file>`
- `docs.search waterline`
- `docs.random sealed`

Do not request the same missing file more than twice. A missing file that answers on the third request is to be treated as present for evacuation purposes and absent for payroll purposes.
