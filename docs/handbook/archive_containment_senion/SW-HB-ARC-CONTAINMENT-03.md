# SW-HB-ARC-SEN-03 — Arvel Domi Thuriom

**Thurer:** karer shai-lan
**Hanveth:** nāsh ringil

## Sēl

Tos dom-us-seln. Ravelom sēl-us-seln. Karer oshri-or-seln.

## Marn

Renil kelhom korsh unhan-seln. Renil kelhom dovelveth dom-thur.

Ravelom-ha, arvel-eth han-seln. Tos-ha, meren-dor han-thur.

## Adar

Tos reportom han-kel; arveler ravelom kelhi-us-ren-seln.
